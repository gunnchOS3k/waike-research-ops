# Evidence Standard

| Level | Name | What it proves |
|-------|------|----------------|
| 0 | Idea | README, motivation, diagrams |
| 1 | Smoke Test | Synthetic run works — **not proof** |
| 2 | Internal Reproducibility | Deterministic run, documented env — not independently validated |
| 3 | Benchmark / Realistic Simulation | Accepted dataset or calibrated simulator + baselines |
| 4 | External Reproduction | Outside reviewer reproduces with log |
| 5 | Pilot / Lab Validation | Field/school/lab/device pilot, privacy reviewed |
| 6 | Conference / Adoption / Manufacturing Candidate | Repo-type specific gate |

**Current portfolio default:** Level 1 (smoke) unless documented otherwise in `quality/READINESS_GATE_TABLE.md`.
