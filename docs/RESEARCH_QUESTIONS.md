# Research Questions

## Primary questions

1. How does apprenticeship-based learning produce verifiable GitHub research evidence?
2. Can standards-aligned curricula map directly to 6G repo contributions?
3. How do we exceed founder knowledge without unsafe data collection?

## Hypotheses (to validate under supervision)

- H1: Shared site schema enables comparable fairness/energy metrics across nodes.
- H2: AI-RAN policies can improve spectrum/energy tradeoffs vs baselines in simulation.
- H3: Privacy-preserving edge telemetry improves twin calibration feasibility.

## Measurable variables

Spectral efficiency, energy/bit, latency, outage/resilience, Jain fairness, Top-k beam accuracy, dB-loss vs oracle, inference latency, opt-in telemetry rates.

## Scientific contribution (target)

Reproducible open artifacts for community-scale trustworthy AI-native 6G evaluation — not a product claim.

## Societal contribution (target)

Digital equality framing for underserved communities; ethical NTN and remote-first scenarios.

## 3–4 year feasibility

Simulation + open benchmarks + limited opt-in field pilots + apprenticeship pipeline — scoped with supervisor.
