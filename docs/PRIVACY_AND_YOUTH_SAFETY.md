# Privacy and Ethics

## Principles

- **No private payload traffic collection**
- **No unauthorized RF transmission**
- **No minors** without guardian / program approval
- **Opt-in data only**
- **Data minimization**
- **Aggregated reporting only** for public Wi-Fi / RF observation boundaries

## Retention / deletion

- Synthetic data: version-controlled with repo
- Opt-in field logs: delete on participant request; default retention ≤ 90 days unless IRB specifies otherwise

## Public Wi-Fi measurement boundaries

Receive-only / metadata-only where applicable. Document antenna and location consent.
