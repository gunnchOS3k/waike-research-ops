# Embedded Systems and Device Prototyping

## Course purpose

Prepare WAIKE learners for industry-ready competence in **embedded systems and device prototyping**, aligned with the gunnchOS3k MLV ecosystem and gunnchAI3k tutoring.

## Who it is for

- Level 1–4 learners (see `docs/04_WAIKE_LEVELS_0_TO_7.md`)
- Gary UPNOW students, apprentices, and global 7GC campus cohorts

## Prerequisites

See WAIKE level map. Prior digital confidence recommended for beginners.

## Duration options

| Format | Hours | Outcome |
|--------|-------|---------|
| Workshop | 2h | Orientation + one lab |
| Bootcamp | 2 weeks | Portfolio starter |
| Course | 8 weeks | Job-ready fundamentals |
| Apprenticeship | 12 weeks | Capstone + mentor review |

## Learning outcomes

1. Explain core concepts using plain-English intuition (WAIKE Consensus Ladder layer 1).
2. Apply academic foundations with labs (layer 2–3).
3. Map work to industry standards (layer 4–5) — *needs source review for exact objectives*.
4. Connect to gunnchOS research/product where applicable (layer 7–8).

## Weekly outline (8-week default)

- Week 1: Intuition + vocabulary + safety/privacy
- Week 2: Tools setup + first lab
- Week 3–5: Core skills + assessments
- Week 6: Industry/vendor reality week
- Week 7: Research frontier + ethics
- Week 8: Capstone presentation + portfolio

## Hands-on labs

- Lab A: Guided tutorial with gunnchAI3k `/waike lab`
- Lab B: Pair debugging / mentor check-in
- Lab C: Community or product-linked mini-project

## Portfolio artifacts

- README project story
- Screenshot or demo video (no PII)
- Rubric self-assessment + mentor sign-off

## Assessments

- Diagnostic quiz (formative)
- Mastery check (summative, open-book where allowed)
- Capstone demo
